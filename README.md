# Qt-TCP-Chat

## 简介

基于 Qt6 + TCP + MySQL 的局域网即时通讯程序。服务端与客户端分离部署，支持多用户注册登录、一对一实时消息、消息持久化到 MySQL。

**当前状态：🚧 开发中（预计 2026 年 7 月 12 日前完成 1.0）**

---

## 技术栈

| 层级 | 技术 |
|------|------|
| 语言 | C++17 |
| UI 框架 | Qt 6.x (Widgets) |
| 通信协议 | TCP (QTcpSocket / QTcpServer) |
| 数据库 | MySQL 8.0 (QSqlDatabase) |
| 构建工具 | CMake / qmake |
| 版本控制 | Git & GitHub |
